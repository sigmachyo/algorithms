# Algorithms Collection

Коллекция реализаций классических алгоритмов и структур данных на **C++**.

### Стек технологий
- **C++** (100%)
- Стандартная библиотека STL

### Файлы и решаемые задачи
- `arithmetic_progression_missing.cpp` — поиск и восстановление пропущенного элемента в арифметической прогрессии
- `comparison_stupid_shell.cpp` — сравнение "глупой" сортировки и сортировки Шелла
- `md5_hash_table.cpp` — хеш-таблица с открытой адресацией и хешированием MD5
- `search_sequential_interpolation.cpp` — последовательный + интерполяционный поиск
- `sum_of_factorial.cpp` — сумма факториалов (итеративно + рекурсивно)
- `trees_remove_less_than_half_max.cpp` — удаление узлов меньше половины максимального в бинарном дереве

### Смысл проекта
Демонстрация понимания фундаментальных алгоритмов и структур данных. Каждый файл — самостоятельное решение конкретной задачи.

- Подготовка к собеседованиям и олимпиадам
- Портфолио алгоритмических навыков
- Обучение и закрепление материала по курсу "Алгоритмы и структуры данных"
